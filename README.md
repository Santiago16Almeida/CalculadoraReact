# Calculadora Básica en React Native (Expo)

¡Hola! Este es mi primer proyecto móvil desarrollado con **React Native** y **Expo**. Es una calculadora funcional que realiza operaciones aritméticas básicas.


## Características
* Suma
* Resta
* Multiplicación
* División (con validación para no dividir entre cero)
* Botón para limpiar los campos
* Teclado numérico optimizado para dispositivos móviles


## Tecnologías Utilizadas
* **React Native** (Componentes nativos y Hooks como `useState`)
* **Expo** (Para un entorno de desarrollo rápido)
* **JavaScript** (Lógica de las operaciones)


## Instalación y Uso

Si quieres probar este proyecto en tu computadora, sigue estos pasos:

1. **Clona este repositorio:**

   git clone [https://github.com/Santiago16Almeida/CalculadoraReact.git](https://github.com/Santiago16Almeida/CalculadoraReact.git)

2. **Entra a la carpeta del proyecto:**

    cd CalculadoraReact

3. **Instala dependencias:**

    npm install

4. **Inicia el servidor de Expo:**

    npx expo start

5. **Abre la app Expo Go en tu celular (Android o iOS) y escanea el código QR que aparece en la terminal**
